## ğŸ› Quick Troubleshooting

### Console won't load
```powershell
# Check server is running
Get-Process -Name "node"

# Restart server
.\stop-console.ps1
.\start-console.ps1
```

### Data not showing
```
1. Open browser console (F12)
2. Check for red errors
3. Verify API calls return 200 status
4. Check database path in server.js
```

### Charts not rendering
```
1. Wait 3-5 seconds after page load
2. Click "Refresh Data" button
3. Check browser console for Chart.js errors
4. Verify Chart.js CDN loaded (Network tab)
```

### Slow performance
```
1. Close unused tabs in console
2. Reduce operations log query limit
3. Clear browser cache
4. Restart console server
```
