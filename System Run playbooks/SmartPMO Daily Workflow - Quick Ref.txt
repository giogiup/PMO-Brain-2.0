## üìä Verification Queries

**Check Today's Progress:**
```sql
SELECT 
    COUNT(*) as total,
    SUM(CASE WHEN pmo_score IS NOT NULL THEN 1 ELSE 0 END) as scored,
    SUM(CASE WHEN content_fetched = 1 THEN 1 ELSE 0 END) as fetched,
    SUM(CASE WHEN keywords_extracted = 1 THEN 1 ELSE 0 END) as keywords,
    SUM(CASE WHEN newsletter_created = 1 THEN 1 ELSE 0 END) as newsletter
FROM daily_insights
WHERE published_date = DATE('now')
```

**View Top Scores:**
```sql
SELECT id, title, pmo_score, fetch_status
FROM daily_insights 
WHERE published_date = DATE('now') AND pmo_score >= 70
ORDER BY pmo_score DESC
```

---

## üîß Key Database Fields

**Daily Insights Table:**
- `published_date` - Article publish date (NOT discovery date)
- `discovered_at` - When we found the article
- `pmo_score` - Relevance score (1-100)
- `content_fetched` - Boolean flag
- `fetch_status` - 'success', 'failed', 'pending'
- `full_content` - Complete article text
- `keywords_extracted` - Boolean flag
- `newsletter_created` - Boolean flag
